## 这是什么

一个**经过策展**的 Claude Code 配置注册表——不是又一个"塞 2000 个 skill"的大杂烩,而是替中文开发者**挑出真值得装的那几个**,每个都过三道闸:

- ✅ **合规** — license 兼容、署名原作者、留 takedown 通道
- 🔒 **安全** — 审过代码,不破坏 / 不外传 / 不藏后门(装得放心)
- 🎯 **品味** — 真解决问题,"轻而狠"优于 800 行瑞士军刀

> 我们相信生态共识:**AI 缺的不是能力,是纪律;而纪律可以用纯文本分发。**

## 怎么用

候选库里标 🟢 的是上架候选,标 🔵 的是信源(只读找线索)。看中哪个,顺着来源仓自取——我们标了 license 和安全红旗,帮你避坑。

## 一点原则

- **来源不限,合规与安全零容忍,品味从宽**。
- **进货单位是单品,不是仓**——大礼包只当信源,我们穿透进去挑单颗珍珠。
- 每个 license 标注:`✅亲验` = 我们 clone 核对过;`⏳称` = 仅来源声称,以原仓为准。
